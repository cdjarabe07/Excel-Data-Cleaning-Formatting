\# Excel Data Cleaning \& Formatting Project



\## Project Objective



The objective of this project is to clean and organize a retail sales dataset using Python and Excel-oriented data cleaning techniques.



The project focuses on:

\- Handling missing values

\- Removing duplicate records

\- Standardizing data formats

\- Correcting inconsistent values

\- Preparing the dataset for future analysis



\---



\## Tools Used



\- Python

\- Pandas

\- Jupyter Notebook

\- Matplotlib



\---



\## Dataset



The dataset contains retail store transaction data including:

\- Product categories

\- Payment methods

\- Quantities purchased

\- Total spending

\- Transaction dates

\- Store locations



The original dataset intentionally contained:

\- Missing values

\- Duplicates

\- Inconsistent formatting

\- Incorrect totals



\---



\## Data Cleaning Process



\### 1. Dataset Inspection

\- Checked dataset structure

\- Identified missing values

\- Reviewed data types



\### 2. Missing Value Handling

\- Filled numerical missing values using median

\- Filled categorical missing values using mode



\### 3. Duplicate Removal

\- Removed duplicate rows to improve data quality



\### 4. Data Standardization

\- Standardized text formatting

\- Converted date columns to proper datetime format



\### 5. Data Validation

\- Checked for negative values

\- Corrected inconsistent transaction totals



\---



\## Basic Analysis



A simple exploratory analysis was performed after cleaning:

\- Most common product categories

\- Average customer spending

\- Most frequently used payment methods



\---



\## Visualization



The project includes a basic visualization showing transaction distribution by category.



\---



\## Final Result



The dataset was successfully cleaned and transformed into a structured and analysis-ready format.
